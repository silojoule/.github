# Silojoule

**Universal open-hardware platform for accessible energy autonomy.**

Energy from any source, into any battery, for any DC load — coordinated over an open 48 V DC bus, independent of grid infrastructure and vendor clouds.

🔌 **Polymorphic hardware** — one converter module, five operating roles, configured by firmware
🔓 **Strong copyleft** — CERN-OHL-S v2 (hardware) + GPL v3 (firmware): forkable forever
📴 **Offline-first** — every core function runs without cloud, account, or internet
🔋 **Chemistry-agnostic** — open charge algorithms for any battery, including second-life packs

→ Main repository: [silojoule-open](https://github.com/silojoule/silojoule-open)
→ Website: [silojoule.org](https://silojoule.org)
